# Git Commands

## 1. Repository Commands
- `git init` – Initialize a new Git repository
- `git clone <repo_url>` – Clone an existing repository

## 2. Staging Commands
- `git add <file>` – Add a specific file to staging area
- `git add .` – Add all files to staging area

## 3. Commit Commands
- `git commit -m "message"` – Commit changes with a message

## 4. Status and History
- `git status` – Check the status of files
- `git log` – View commit history

## 5. Branch Commands
- `git branch` – List branches
- `git branch <branch_name>` – Create a new branch
- `git checkout <branch_name>` – Switch to another branch
- `git checkout -b <branch_name>` – Create and switch to a new branch

## 6. Remote Repository Commands
- `git remote -v` – Show remote repositories
- `git push origin <branch_name>` – Push changes to remote repository
- `git pull origin <branch_name>` – Pull changes from remote repository
- `git fetch` – Fetch updates from remote repository

## 7. Merge Commands
- `git merge <branch_name>` – Merge a branch into the current branch

## 8. Undo Commands
- `git reset <file>` – Unstage a file
- `git revert <commit_id>` – Undo a commit by creating a new commit
